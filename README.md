## Description

[Nest](https://github.com/nestjs/nest) boilerplate with authentication.

## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Test

```bash
# e2e tests
$ pnpm run test
```
